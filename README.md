# Web Calculator

A minimal, accessible web calculator with keyboard support.

- No frameworks, just HTML/CSS/JS
- Keyboard-friendly: 0–9, + - * /, Enter, Backspace
- Accurate to 12 decimal places (rounded)

Quick start: open index.html in a browser.
